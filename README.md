## Summary
![Active Directory](https://img.shields.io/badge/Active_Directory-5E5E5E?style=for-the-badge&logo=microsoft&logoColor=white)

In this lab, I demonstrated how to create a new partition on a Windows Server and configure it as a shared network drive. The shared location was intended to store resources that could be deployed to client computers, such as software packages and desktop wallpaper files.

## Goal

Create a centralized network share that can be accessed by client systems for software deployment and file distribution.

## What I Did

1. Created a new partition on the server.
2. Configured the partition for shared access.
3. Created a network share on the newly partitioned drive.
4. Prepared the shared location to host deployment resources, including software installers and desktop wallpaper files.
5. Demonstrated how the shared drive could be used as a centralized repository for client computer resources.

## Tools and Technologies

* Windows Server 2025
* Disk Management
* Shared Folders (SMB Network Shares)

## Technical Skills Demonstrated

* Storage and partition management
* Network share configuration
* Windows Server administration
* Centralized resource distribution
* File sharing and access management

## Outcome

The lab resulted in a dedicated shared storage location that can be used to distribute files and resources to client computers within the environment.
